# Open Source Repositories and Contributions

> Last Updated: 2026-02-11  

## 1. Repository Links

- API (Main Project): https://github.com/dujiao-next/dujiao-next
- User (Frontend): https://github.com/dujiao-next/user
- Admin (Backend): https://github.com/dujiao-next/admin
- Document (Documentation): https://github.com/dujiao-next/document

## 2. Terminology

- "User" in the documentation refers to the frontend project (repository name: `user`).
- If your local folder still uses the old name `web`, just replace `user` with `web` in commands.

## 3. PR Submission Suggestions

- Feature development: It is recommended to include usage instructions and screenshots (for UI-related changes).
- Bug fixes: It is recommended to provide reproduction steps, your fix approach, and verification results.
- Documentation additions: You are welcome to directly modify the `Document` repository and submit a PR.

## 4. Recommended Issue Template

When creating an Issue, please include:

- Problem description
- Steps to reproduce
- Expected result
- Actual result
- Environment (system, Node/Go version)

Thanks to every contributor.
