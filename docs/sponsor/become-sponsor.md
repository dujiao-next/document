# 成为赞助商

> 更新时间：2026-02-12

感谢你对 Dujiao-Next 开源项目的支持 ❤️

赞助费用将用于：

- 持续维护与修复问题
- 新功能开发与文档完善
- 社区支持与版本发布

## 1. 成为赞助商的好处

成为赞助商后，你可以获得以下曝光与权益：

- 在官方文档首页展示品牌广告位（按赞助等级）
- 在文档右侧目录区域展示广告位（按赞助等级）
- 在赞助模块中展示你的品牌名称、Logo、简介与跳转链接
- 项目方优先响应赞助商反馈（功能建议/问题沟通）

## 2. 赞助商级别

> 以下为标准赞助档位，若需定制合作方案可单独沟通。

| 级别 | 赞助费（月）      | 广告位权益           | 说明 |
| --- |-------------|-----------------| --- |
| 铂金赞助商（Platinum） | 300 USD / 月 | 首页赞助区 + 右侧目录广告位 | 最佳曝光，推荐品牌合作 |
| 黄金赞助商（Gold） | 200 USD / 月 | 右侧目录广告位         | 稳定曝光，适合长期投放 |
| 白银赞助商（Silver） | 100 USD / 月 | 文档赞助商目录         | 入门级品牌展示 |

### 广告素材建议

- 图片比例建议：16:9 或等比例品牌 Logo 图
- 文案建议：标题 + 一句简短描述
- 跳转链接建议：官网主页 / 活动页 / 产品页
- 无图片素材时，可降级为“标题 + 描述”纯文字展示
