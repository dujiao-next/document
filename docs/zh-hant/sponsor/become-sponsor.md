# 成為贊助商

> 更新時間：2026-02-12

感謝你對 Dujiao-Next 開源項目的支持 ❤️

贊助費用將用於：

- 持續維護與修復問題
- 新功能開發與文檔完善
- 社區支持與版本發佈

## 1. 成為贊助商的好處

成為贊助商後，你可以獲得以下曝光與權益：

- 在官方文檔首頁展示品牌廣告位（按贊助等級）
- 在文檔右側目錄區域展示廣告位（按贊助等級）
- 在贊助模塊中展示你的品牌名稱、Logo、簡介與跳轉連結
- 項目方優先響應贊助商反饋（功能建議/問題溝通）

## 2. 贊助商級別

> 以下為標準贊助檔位，若需定製合作方案可單獨溝通。

| 級別 | 贊助費（月）      | 廣告位權益           | 說明 |
| --- |-------------|-----------------| --- |
| 鉑金贊助商（Platinum） | 300 USD / 月 | 首頁贊助區 + 右側目錄廣告位 | 最佳曝光，推薦品牌合作 |
| 黃金贊助商（Gold） | 200 USD / 月 | 右側目錄廣告位         | 穩定曝光，適合長期投放 |
| 白銀贊助商（Silver） | 100 USD / 月 | 文檔贊助商目錄         | 入門級品牌展示 |

### 廣告素材建議

- 圖片比例建議：16:9 或等比例品牌 Logo 圖
- 文案建議：標題 + 一句簡短描述
- 跳轉連結建議：官網主頁 / 活動頁 / 產品頁
- 無圖片素材時，可降級為“標題 + 描述”純文字展示
