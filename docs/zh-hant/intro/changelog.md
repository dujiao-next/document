# 更新日誌

## [v0.0.3-beta] - 2026-02-22

### 更新內容：
- 新增用戶錢包體系：支援餘額、充值、錢包明細流水記錄（含充值、支付、退款回退等資金動向）。
- 升級訂單支付能力：支援僅餘額支付、餘額+線上支付組合支付、僅線上支付；訂單未支付成功或取消時自動退回已扣餘額。
- 後台新增錢包管理能力：支援查看用戶餘額與明細、手動增減用戶餘額、訂單退款僅退回用戶餘額。
- 新增 Telegram 登錄能力：首次 Telegram 登錄且未綁定帳號時自動建立帳號並直接登錄。
- 安全中心新增 Telegram 綁定/解除綁定：已登錄用戶可綁定或解除綁定 Telegram；未綁定真實郵箱的 Telegram 新帳號不允許解除綁定。
- 完善帳號安全策略：新增 `email_change_mode` 與 `password_change_mode` 返回字段，支援「僅綁定新郵箱」與「首次設置密碼無需舊密碼」流程。
- 優化用戶中心導覽：左側選單順序調整為「概覽 / 我的訂單 / 我的錢包 / 安全中心 / 資料設定」，並補充選單圖標。
